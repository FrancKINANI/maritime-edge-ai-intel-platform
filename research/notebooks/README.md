# Phase 0 Notebooks

Jupyter notebooks for the full Phase 0 evaluation pipeline, designed to run in Google Colab.

## Notebooks

| Notebook | Purpose |
|----------|---------|
| `colab_research_pipeline_v1.ipynb` | End-to-end pipeline: download → preprocess → detect → evaluate |
| `colab_traceability_check_v1.ipynb` | Verify reproducibility between local and Colab environments |
| `colab_research_traceability_check_v2.ipynb` | Verify traceability of results of the research step by step through the pipeline execution |

## Usage

Open in Google Colab or run locally with Jupyter:

```bash
jupyter notebook research/notebooks/
```
